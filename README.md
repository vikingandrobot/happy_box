# happy_box
A happy box that opens up and says hello when you walk in front of it !


Components:
- Arduino or similar
- Breadboard
- Movement sensor 
- Electrical motor
- On/Off Switch 
- Battery
- USB Charger & Port
- Audio Input/output
- ...


Next level: 
- Internet/Bluetooth Connection 
- App to record and store audio files in happy box remotely
- ...
